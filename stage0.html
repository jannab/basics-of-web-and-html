Getting Started with HTML

How the Web Works

The web consists of a lot of computers and servers all over the world. When someone tries to reach a web page like udacity.com, the computer actually sends a HTTP Request to a server. The server looks up the right HTML document and sends it back to the computer. Then, the browser of the person interprets the HTML code and shows the web page accordingly.

I didn't learn any new concepts here, but this was a great quick review, which was definitely not boring.

HTML

HTML is an abbreviation for Hypertext Markup Language. Even though the web contains videos, pictures and much more, most of it is written in HTML and videos and pictures are embedded in the HTML code. In HTML we have text, that describes the texts you see and the markup describes the way it looks.

This was a great overview. It helped me refresh the basic HTML knowledge I have.

Tags and Elements

Elements in HTML are defined using tags. An HTML document consits of several elements. A title for example is an element and we would use a h1 tag for that.

Inline vs Block Elements

Block elements have an "invisible box" around them. Therefore, block elements create new paragraphs. Inline elements can be in a paragraph without having to start a new one.
